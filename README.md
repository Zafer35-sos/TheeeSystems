# System Hunter - PWA Deployment Guide

This app is ready for conversion to an Android APK via PWABuilder.

## Deployment Steps

1. **GitHub Setup**:
   - Create a new repository on GitHub.
   - Run `git init`, `git add .`, `git commit -m "Initial commit"`.
   - Push your code: `git remote add origin YOUR_REPO_URL` and `git push -u origin main`.

2. **Firebase App Hosting**:
   - Go to [Firebase Console](https://console.firebase.google.com/).
   - Select **App Hosting** from the Build menu.
   - Connect your GitHub account and select your repository.
   - Choose the branch (e.g., `main`).
   - Firebase will automatically deploy your Next.js app.

3. **APK Generation**:
   - Once the deployment is live, copy your domain (e.g., `https://my-hunter-app.web.app`).
   - Go to [PWABuilder.com](https://www.pwabuilder.com/).
   - Paste your URL and click **Start**.
   - Download the generated Android package and build your APK.

## Features
- **ORV Style Messages**: Majestic notifications for Level-Ups and Title Unlocks.
- **Multimodal Perception**: Upload photos to the Terminal for AI analysis.
- **Background Reminders**: Temporal alerts that notify your device even if the app is minimized.
- **Dynamic XP Scaling**: AI evaluates your tasks (e.g., 50 pages = 75 XP).
