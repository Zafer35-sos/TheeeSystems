# **App Name**: Ascension Log

## Core Features:

- User Profile & Progression: Display user's current level, experience points (XP) towards the next level, and an editable title.
- AI Daily Task Generator: Leverage a generative AI tool to suggest and create relevant daily tasks based on a user's general goal, assigning appropriate XP rewards.
- Daily Task Management: Allow users to view, add, mark as complete, and reset their daily tasks.
- XP and Leveling System: Automatically calculate and apply XP gained from completed tasks, and trigger a level-up when sufficient XP is accumulated.
- Progress Tracking Dashboard: Display daily progress including number of tasks completed, tasks remaining, and XP gained for the current day.
- User Data Persistence: Securely store user profiles, task lists, and progression data across sessions.

## Style Guidelines:

- A dark color scheme suitable for an immersive 'system' interface. The primary color for interactive elements and highlights is a vibrant yet rich violet, 'Accent Amethyst' (#732AFF). The background color, 'Midnight Aura' (#221E29), is a deep desaturated purplish-grey to maintain visual continuity. A cool, contrasting blue, 'System Glow' (#5AB8FF), is used for secondary accents or status indicators.
- For readability and a modern, neutral 'system' feel, the font 'Inter' (sans-serif) will be used for both headlines and body text.
- Use clean, minimalistic line-art icons that resemble a holographic UI, maintaining the futuristic and 'system' aesthetic seen in the reference image. Icons for quests, stats, system, and task actions should be easily recognizable.
- The layout will feature clearly delineated sections and cards for task lists and progress metrics, mimicking the segmented dashboard in the provided screenshot. Ample use of padding and negative space to ensure clarity and ease of navigation. Key information like level and XP will be prominently displayed at the top.
- Implement subtle and quick animations for user feedback, such as task completion and level-ups, enhancing the 'system notification' feel. Smooth transitions between tabs (Quests, Stats) and slight glows on interactive elements will add to the immersive experience.
